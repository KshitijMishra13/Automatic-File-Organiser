# Automatic-File-Organiser
## File Organizer Automation

A Python automation project that automatically organizes files into folders based on their file type.

## Features
- Automatically sorts files by extension
- Real-time folder monitoring using `watchdog`
- Creates folders automatically
- Supports PDFs, Excel files, ZIPs, Images, Text files, and more
- Generates file movement logs

## Technologies Used
- Python
- os
- shutil
- watchdog

## How to Run

Install dependencies:

```bash
pip install watchdog
